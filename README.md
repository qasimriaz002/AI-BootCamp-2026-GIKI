## Phase 1 — Python Foundations

A comprehensive Python programming course designed for the **GIKI AI Boot Camp 2026**.

This repository contains all supporting resources required to build a strong Python foundation before moving into Machine Learning, Deep Learning, Computer Vision, NLP, and Agentic AI.

---

## 🎯 Learning Objectives

After completing this phase, students will be able to:

- Write clean and readable Python programs
- Understand Python syntax and programming fundamentals
- Solve problems using functions and modular programming
- Work with files and exceptions
- Manipulate data using NumPy
- Analyze structured datasets using Pandas
- Create professional visualizations using Matplotlib
- Develop a strong programming foundation for AI and Machine Learning

